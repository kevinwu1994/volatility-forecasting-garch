# Project Report

This folder contains the final report for the project:

**Volatility Forecasting and Risk Management Using GARCH Models**

## File

- 📄 [Final Project Report](./volatility-forecasting-garch-report.pdf)

## What the report covers

The report includes:

- Introduction and research objectives
- Data description and stylized facts of ETF returns
- Volatility modeling using GARCH(1,1), EGARCH, and GJR-GARCH
- Out-of-sample forecast evaluation using MSE, MAE, and QLIKE
- Value-at-Risk (VaR) construction and backtesting using the Kupiec test
- Risk management discussion and practical implications
- Limitations of the modeling framework
- Final conclusions and future improvement directions

## Main takeaway

The report shows that the simpler GARCH(1,1) model delivers the most robust out-of-sample forecasting performance and can be effectively translated into reliable VaR-based risk estimates.
