# Presentation Slides

This folder contains the presentation slides for the project:

**Volatility Forecasting and Risk Management Using GARCH Models**

---

## File

- [Download Slides](./volatility-forecasting-garch-slides.pptx)

---

## Content Overview

The slides present the full analytical workflow:

- Motivation and stylized facts of financial returns  
- Volatility modeling using GARCH-family models  
- Out-of-sample forecasting framework  
- Model evaluation using MSE, MAE, and QLIKE  
- Value-at-Risk (VaR) construction  
- Backtesting using the Kupiec test  
- Risk management insights and limitations  

---

## Purpose

The presentation summarizes the key methodology and findings of the project, with a focus on practical risk management applications.

---

## Key Takeaway

A simple GARCH(1,1) model provides robust volatility forecasts and produces reliable VaR estimates validated through statistical backtesting.
